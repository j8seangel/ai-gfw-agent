# GFW Agents playground

## Install Dependencies

```bash
npm install
```

## Install Ollama

To install on macOS:

```bash
curl -fsSL https://ollama.com/install.sh | sh
```

Install at least one model, currently using [granite3.3](https://ollama.com/library/granite3.3)

```bash
ollama run granite3.3
```

## Run the project

```bash
npm run dev
```
